Modular Gun Pack - A fully customizable set of gun assets with which you can create your own firearms. Includes tool that can be used to easily select different elements and save the finished piece as a prefab.

Includes:

- 3 Frames
- 3 Barrels
- 2 Triggers
- 2 Stocks
- 3 Scopes
- 2 Magazines
- 1 Bullet type

Every module has both a high and low poly version.

To use the tool, just create an empty GameObject and add "GunPackScript" to it.
